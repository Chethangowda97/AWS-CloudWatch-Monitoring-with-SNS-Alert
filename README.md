# AWS-CloudWatch-Monitoring-with-SNS-Alert
Project Overview

This project demonstrates how to monitor an Amazon EC2 instance using AWS CloudWatch and configure an alert system that sends email notifications using Amazon SNS when CPU utilization exceeds a defined threshold.

The monitoring system provides real-time visibility into EC2 performance and helps administrators respond quickly to high resource usage.
Services Used
-Amazon EC2
-Amazon CloudWatch
-Amazon SNS
-4IAM

Project Objective
-Launch an EC2 instance
-Monitor EC2 metrics using CloudWatch
-Create a CloudWatch dashboard
-Configure a CloudWatch alarm for CPU utilization
-Send email alerts through SNS when CPU utilization exceeds 5%
